About Lens Simulator

Explore how light rays interact with different types of lenses and understand image formation through ray diagrams.

Types of Lenses:

  • Convex Lens: Converges light rays to a focal point
  • Concave Lens: Diverges light rays from a focal point

Key Concepts:

  • Focal Point: The point where parallel light rays converge or appear to diverge from
  • Focal Length: The distance between the lens and the focal point
  • Object Distance: The distance between the object and the lens
  • Image Distance: The distance between the image and the lens
  • Magnification: The ratio of image height to object height

Lens Equation:

1/f = 1/do + 1/di

where:

  • f = focal length
  • do = object distance
  • di = image distance

Magnification Equation:

M = -di/do = hi/ho

where:

  • M = magnification
  • hi = image height
  • ho = object height
